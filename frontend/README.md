# 🌱 AgroAI – Smart Farming Assistant

AgroAI is an AI-powered web application that helps farmers detect crop diseases instantly using image analysis and provides smart recommendations for treatment.

---

## 🚀 Features

- 🌿 AI-based crop disease detection (Gemini API)
- 📸 Upload crop/leaf image for analysis
- 💊 Smart solution & prevention tips
- 🌐 Multi-language support (Hindi, English, Marathi)
- 🔊 Voice output for farmers
- 📊 Graph visualization of results
- 🤖 Built-in chatbot for farming help

---

## 🧠 How It Works

1. Upload crop image  
2. Click **Analyze**  
3. Image sent to backend  
4. Gemini AI analyzes image  
5. Result + solution displayed  
6. Graph updates dynamically
